# 妈妈的菜谱 — 项目说明

## 项目目标

暑假跟妈妈学做菜，每天记录一道菜，形成结构化的家传菜谱库。
视频是素材来源，最终产物是可查阅、可复现的菜谱文档。

## 架构说明

**代码与数据分离**：
- `mom-recipes/` — 纯代码，Electron 应用
- `E:\obsidian\生活笔记\` — Obsidian Vault，全部数据

应用只是 CRUD 界面，所有菜谱数据存放在 Obsidian Vault 中，方便在 Obsidian 中使用 Dataview 插件进行动态查询。

## 数据流

```
Electron 应用 (CRUD)  ──读写──▶  Obsidian Vault (数据)
       │                              │
       │  新建菜谱 → 写 .md 到分类目录
       │  编辑菜谱 → 覆写 .md
       │  删除菜谱 → 删 .md
       │  记做饭日志 → 追加到 做饭日志.md
       │  同步索引 → 扫描全部菜谱 → 重写 菜谱索引.md
       │                              │
       └── 今天吃啥 ← 读索引 → 按上次做排序推荐
```

## 菜谱文件格式

每个菜谱 .md 文件顶部有 YAML frontmatter：

```yaml
---
状态: 已学会
难度: ⭐
分类: 炒
学会日期: 2026-07-01
用时: 15
份量: 2
上次做: 2026-01-25
评分: 8
---
```

应用通过解析 frontmatter 获取元数据，不再依赖 README.md 管理状态。

## 功能说明

| 功能 | 说明 |
|------|------|
| 菜谱管理 | 新建、编辑、删除菜谱 |
| 做饭日志 | 记录每次做菜的评分、问题、改进 |
| 今天吃啥 | 按"上次做"日期推荐很久没做的菜 |
| 索引同步 | 自动生成 `菜谱索引.md`（含 Dataview 查询） |
| 视频管理 | 导入、查看做菜视频 |

## 工作流

1. 白天跟妈妈学做菜，拍视频（5-8 段，总共 3-5 分钟素材）
2. 晚上用剪映剪成 1-2 分钟精华版，存到 `视频素材/DayXX-菜名/`
3. 用 Electron 应用新建菜谱（自动生成带 frontmatter 的 .md 文件）
4. 在应用中编辑菜谱内容
5. 做完饭后记录做饭日志
6. 定期同步索引

## 记录原则

- 精确用量，不写"适量"（用秤量，记具体数字）
- 火候写感官信号（视觉/听觉/颜色），不只写"中火"
- 妈妈的秘诀单独列出来
- 踩过的坑也要记
- 视频重点拍手部操作手法

## 命名规范

- 菜谱文件：`菜名.md`（如 `红烧肉.md`）
- 视频文件夹：`DayXX-菜名/`（如 `Day05-红烧肉/`）
- 剪辑成品：`菜名.mp4`

## Obsidian 端配置

### 必装插件
- **Dataview**：动态查询菜谱

### 可选插件
- **Templater**：菜谱模板
- **Calendar**：做饭日志按日历查看

### Dataview 查询示例

**很久没做的菜**：
```dataview
TABLE 难度, 用时, 上次做 AS "上次做", 评分
FROM "菜谱"
WHERE 状态 = "已学会"
SORT 上次做 ASC
```

**按分类统计**：
```dataview
TABLE length(rows) AS "数量"
FROM "菜谱"
GROUP BY 分类
```

**高分菜谱**：
```dataview
TABLE 难度, 用时, 评分
FROM "菜谱"
WHERE 评分 >= 8
SORT 评分 DESC
```
