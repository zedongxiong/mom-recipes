# 妈妈的菜谱 — 项目说明

## 项目目标

暑假跟妈妈学做菜，每天记录一道菜，形成结构化的家传菜谱库。
视频是素材来源，最终产物是可查阅、可复现的菜谱文档。

## 项目结构

```
mom-recipes/
├── CLAUDE.md              ← 本文件
├── README.md              ← 总目录索引（已学菜谱列表）
├── 菜谱模板.md            ← 每道菜的标准化记录模板
├── 菜谱/                  ← 按分类存放菜谱文档
│   ├── 肉类/
│   ├── 海鲜/
│   ├── 蔬菜/
│   ├── 汤品/
│   ├── 主食/
│   ├── 凉菜/
│   └── 节日菜/
└── 视频素材/              ← 按日期存放原始视频
    └── Day01-菜名/
```

## 工作流

1. 白天跟妈妈学做菜，拍视频（5-8 段，总共 3-5 分钟素材）
2. 晚上用剪映剪成 1-2 分钟精华版，存到 `视频素材/DayXX-菜名/`
3. 按 `菜谱模板.md` 填写菜谱文档，存到对应分类文件夹
4. 更新 `README.md` 索引

## 记录原则

- 精确用量，不写"适量"（用秤量，记具体数字）
- 火候写感官信号（视觉/听觉/颜色），不只写"中火"
- 妈妈的秘诀单独列出来
- 踩过的坑也要记
- 视频重点拍手部操作手法

## 命名规范

- 菜谱文件：`菜名.md`（如 `红烧肉.md`）
- 视频文件夹：`DayXX-菜名/`（如 `Day05-红烧肉/`）
- 剪辑成品：`菜名.mp4`
